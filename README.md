# Distributed Cache with Consistency Guarantees
Redis-like distributed cache with configurable consistency, replication, and partitioning.
## Features
- Configurable consistency (Eventual/Strong/Quorum)
- Multiple eviction policies (LRU/LFU/FIFO/TTL)
- Automatic replication across nodes
- Consistent hashing for partitioning
## Tech Stack
Java 17, Spring Boot 2.7.11, Redis
